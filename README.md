## 🏃‍♀️ Run locally

### Clone the project

`git clone https://github.com/dboinska/envelo-paczkomat`

### Go to the project directory

`cd envelo-paczkomat`

### Install dependencies

`npm install`

### Start the server

`npm run dev`
